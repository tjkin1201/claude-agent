# 창작 & 디자인 전문 워크플로우

## 🎨 브랜드 디자이너 워크플로우

### 브랜딩 프로젝트 전 과정
```yaml
research_phase:
  - brand-strategist: "브랜드 전략 및 포지셔닝"
  - market-analyzer: "경쟁사 및 트렌드 분석"
  - target-researcher: "타겟 오디언스 분석"

concept_phase:
  - concept-developer: "브랜드 컨셉 개발"
  - mood-board-creator: "비주얼 무드보드"
  - naming-specialist: "브랜드명 및 슬로건"

design_phase:
  - logo-designer: "로고 디자인 및 변형"
  - visual-identity-designer: "비주얼 아이덴티티 시스템"
  - typography-specialist: "타이포그래피 시스템"

application_phase:
  - brand-guideline-creator: "브랜드 가이드라인"
  - application-designer: "브랜드 적용 디자인"
  - brand-asset-manager: "브랜드 에셋 관리"
```

### 실행 명령어
```bash
# 브랜딩 프로젝트 시작
/workflow start --category "creative" --type "branding" --industry "tech"

# 로고 디자인만 집중
/workflow focus --task "logo-design" --iterations 5

# 브랜드 가이드라인 생성
/workflow generate --deliverable "brand-guidelines" --format "digital"
```

## 📹 콘텐츠 크리에이터 워크플로우

### 유튜브/소셜미디어 콘텐츠 제작
```yaml
planning_phase:
  - content-strategist: "콘텐츠 전략 및 캘린더"
  - trend-researcher: "트렌드 분석 및 키워드"
  - audience-analyzer: "오디언스 분석"

creation_phase:
  - script-writer: "대본 작성"
  - video-producer: "영상 기획 및 촬영"
  - thumbnail-designer: "썸네일 디자인"

production_phase:
  - video-editor: "영상 편집 및 후처리"
  - audio-engineer: "오디오 최적화"
  - graphic-animator: "그래픽 및 애니메이션"

distribution_phase:
  - seo-optimizer: "검색 최적화"
  - social-media-manager: "소셜미디어 배포"
  - community-manager: "커뮤니티 관리"

analytics_phase:
  - performance-analyst: "성과 분석"
  - audience-insights: "오디언스 인사이트"
  - growth-optimizer: "성장 최적화"
```

### 배치 제작 최적화
```bash
# 주간 콘텐츠 배치 제작
/workflow batch --weekly --content-type "educational" --platforms "youtube,instagram"

# 시리즈 콘텐츠 기획
/workflow series --episodes 10 --theme "AI기초" --target-length "10min"

# 크로스플랫폼 최적화
/workflow optimize --source "youtube" --adapt-to "tiktok,instagram,twitter"
```
