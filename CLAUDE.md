### MCP Server Integration
**자동 서버 선택**: 작업에 최적화된 MCP 서버가 자동으로 활성화됩니다.

- **Context7** (`--c7`): 공식 라이브러리 문서, 프레임워크 패턴
- **Sequential** (`--seq`): 복잡한 분석, 체계적 사고
- **Magic** (`--magic`): UI 컴포넌트 생성, 디자인 시스템
- **Playwright** (`--play`): E2E 테스팅, 브라우저 자동화
